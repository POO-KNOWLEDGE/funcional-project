# Casos de uso del menú


## Registrar Cliente

El usuario puede registrar un cliente ingresando:

- Nombre
- Documento
- Teléfono
- Código de cliente


## Registrar Robot

El sistema permite registrar robots indicando:

- Serial
- Marca
- Cantidad de sensores


## Registrar Dron

El sistema permite registrar drones indicando:

- Serial
- Marca
- Altura máxima


## Buscar Equipo

El usuario puede buscar equipos mediante el serial.


## Reparar Equipo

El sistema cambia el estado del equipo a “Reparado”.


## Mostrar Órdenes

El sistema muestra las órdenes registradas.
